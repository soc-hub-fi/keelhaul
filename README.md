# register_testing2

Simple skeleton for register test case generation.

## How to use

Copy your SVD-file to path `data/tackle.svd`.

Build testcases.

`just build`

Build and run (**will cause segmentation fault 8-)**).

`just run`

## Dependencies

- `cargo`
    - `curl https://sh.rustup.rs -sSf | sh`
- `just`
    - `cargo install just`
