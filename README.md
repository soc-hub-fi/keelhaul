# register_testing

Simple skeleton for register test case generation.

## How to use

Set path to Python-interpreter to `Justfile`.

Build testcases.

`just build`

Build testcases without parsing.

`just build-without-parsing`

Build and run (**will cause segmentation fault 8-)**).

`just run`

## Dependencies

`python3.10`

`beautifulsoup 4`

`lxml`

`html5lib`

`just`
